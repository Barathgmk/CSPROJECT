## Common Questions

**Q: Where do I configure Reddit API?**  
A: Edit `config/.env` with your Reddit credentials

**Q: Where does the screened stock list save?**  
A: Automatically saves to `data/penny_candidates.csv`

**Q: How do I run the API?**  
A: Just type `python run_api.py` (or `python -m uvicorn src.api:app --host 127.0.0.1 --port 8001`)

**Q: Where's the dashboard?**  
A: Open `frontend/index.html` in your browser

**Q: Can I still use the old commands?**  
A: Yes! The imports are updated, so the code still works

**Q: How do I add new Python modules?**  
A: Put them in `/src/` and import them in `/src/api.py`
